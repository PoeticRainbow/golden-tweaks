# RetroTweaks
Tweak your Minecraft and send it back in time! An old-school tweaks mod that targets modern versions of Minecraft.

RetroTweaks does not replace resource packs. Textures, sounds, etc. are much more appropriate to be included in a resource pack like [Golden Days](https://github.com/PoeticRainbow/golden-days) rather than in a mod.
